## 🚀 Quick Start

### For Members
1. Go to: `/member/pay-penalty`
2. View your unpaid penalties
3. Enter M-Pesa phone number
4. Click "Pay" button
5. Complete M-Pesa payment
6. Done! ✅

**Read**: `PENALTY_WALLET_ACCESS_GUIDE.md` → "Member: How to Pay Penalties"

### For Admin
1. Go to: `/admin/penalty-wallet`
2. View wallet balance
3. Click "Request Withdrawal"
4. Enter amount and reason
5. Submit for approval
6. Wait for signatories to approve

**Read**: `PENALTY_WALLET_ACCESS_GUIDE.md` → "Admin: How to Manage Penalty Wallet"

### For Signatories
1. Go to: `/admin/withdrawal-approval`
2. View pending withdrawals
3. Review amount and reason
4. Click "Approve" or "Reject"
5. Done! ✅

**Read**: `PENALTY_WALLET_ACCESS_GUIDE.md` → "Signatories: How to Approve Withdrawals"
